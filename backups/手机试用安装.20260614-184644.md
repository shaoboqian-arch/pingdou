# BeanDex 手机试用安装

## 当前最快路径

1. 电脑和手机连同一个 Wi-Fi。
2. 电脑运行 `npm run dev:mobile -- --port 5176`。
3. iPhone Safari 打开 `http://192.168.110.227:5176/`。
4. 点分享。
5. 点“添加到主屏幕”。

## iOS 真机包路径

1. 当前已生成 Capacitor iOS 外壳：`ios/App`。
2. 当前已同步 Web 构建产物到 iOS 外壳。
3. 到 Mac 上安装 Xcode 和 CocoaPods。
4. 运行 `npm run cap:sync`。
5. 运行 `npm run cap:open:ios`。
6. Xcode 选择个人 Apple ID 和 iPhone，点 Run。

## 当前限制

Windows 不能执行 `xcodebuild`，不能直接装到 iPhone。
