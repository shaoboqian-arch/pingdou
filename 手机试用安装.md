# BeanDex 手机试用安装

## A. 最快试用：iPhone 局域网访问
1. 电脑和 iPhone 连接同一个 Wi-Fi。
2. 电脑运行：`npm run dev:mobile -- --port 5176`。
3. 确认 Windows 防火墙已放行 TCP `5176`。
4. iPhone Safari 打开：`http://192.168.110.227:5176/`。
5. Safari 分享按钮 -> 添加到主屏幕。

## B. 真机安装：Capacitor iOS
1. 当前 iOS 工程目录：`ios/App`。
2. 当前 App ID：`com.shaobo.beandex`。
3. 当前 App 名称：`BeanDex`。
4. 在 Mac 安装 Xcode、CocoaPods、Node.js。
5. 拷贝整个项目到 Mac。
6. 在项目根目录运行：`npm install`。
7. 运行：`npm run cap:sync`。
8. 运行：`npm run cap:open:ios`。
9. Xcode 选择个人 Apple ID Team。
10. 连接 iPhone，选择设备，点 Run。

## C. Windows 当前限制
Windows 可以构建 Web 和同步 Capacitor 文件，但不能直接运行 `xcodebuild` 安装到 iPhone。
真机安装必须在 Mac + Xcode 上完成。

## D. 到设备前必须确认
1. `npm run build` 通过。
2. `npm run cap:sync` 通过。
3. `ios/App/App/App.xcworkspace` 存在。
4. iPhone 已开启开发者模式。
5. Apple ID 可用于 Xcode 免费签名。
